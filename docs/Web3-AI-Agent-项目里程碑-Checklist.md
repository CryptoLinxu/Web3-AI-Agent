# Web3 AI Agent 项目里程碑 Checklist

## 使用说明

本清单用于跟踪 `Web3 AI Agent` 项目的阶段进展。每一项都应对应真实代码、文档或设计产物，不以口头完成为准。

---

## 阶段 1：项目初始化

- [x] 创建项目基础结构
- [x] 明确技术栈：`Next.js + OpenAI API + ethers + Tailwind`
- [x] 配置环境变量策略
- [x] 明确根目录文档体系
- [x] 建立 `skills/x-ray/` 目录规划

### 阶段产出

- 项目初始化说明
- 文档目录与命名规范
- skill 体系方案文档

### 阶段验收标准

- 文档结构已建立
- 开发方式和阶段流程已写清楚
- skill 体系有明确职责边界

### 关联 skill

- `origin`
- `prd`
- `update-map`

---

- [ ] 完成 MVP PRD
- [ ] 明确目标用户画像

- [ ] 明确 MVP 功能范围
- [ ] 明确非目标
- [ ] 明确风险边界与免责声明
- [ ] 明确验收标准

### 阶段产出

- [`Web3-AI-Agent-PRD-MVP.md`](./Web3-AI-Agent-PRD-MVP.md)
- [`Web3-AI-Agent-阶段执行说明-V3.md`](./Web3-AI-Agent-阶段执行说明-V3.md)

### 阶段验收标准

- PRD 能回答“做什么、不做什么、为什么现在做”
- 文档足以支撑需求拆解
- 无重大范围歧义

### 关联 skill

- `prd`
- `pm`
- `req`

## 阶段 3：Agent 核心能力定义

- [ ] 明确 Agent 的角色定位
- [ ] 明确对话、工具调用、Loop、Memory 的边界
- [ ] 明确哪些能力属于 MVP
- [ ] 明确哪些能力留到后续迭代
- [ ] 明确系统 Prompt 需要承担的职责

### 阶段产出

- Agent 能力定义说明
- MVP 能力边界表

### 阶段验收标准

- 能区分 Chat 与 Agent 的实现差异
- 能清楚说明 Tool、Loop、Memory 的作用
- 不会把非 MVP 能力提前塞入实现

### 关联 skill

- `check-in`
- `architect`
- `audit`
