# Init-Docs

## 适用场景

- 新项目第一次建文档
- 历史文档迁移
- 重建基础索引

## 输入

- 代码库现状
- 现有文档

## 输出

- 初始地图
- 初始索引
- 基础结构化文档

## 流程

1. 扫描项目骨架
2. 建立第一版地图和索引
3. 生成初始文档结构
4. 交给 `update-map`

## 边界

- 不承担具体功能开发
- 不代替后续 `digest`

## 规则

1. 这是 `BOOTSTRAP` 专用 skill。
2. 初始化完成后应交由正常 V3 链路继续演化。
